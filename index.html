Sobre Wit Set

El objetio de Wit Set es englobar todo:

Este proyecto nació como un pasatiempo cuyo objetivo es servir como una plataforma que reune varias aplicaciones web útiles en diferentes tareas, principalmente destinadas a fines de conocimiento general.

Por ejemplo: con Fun Fact Web explora todos los datos curiosos que quieras, o en STEAM Torch actualiza tus conocimientos del modelo educativo STEAM.

Considéralo como una caja de herramientas, solo que no sabes qué tipos de herramientas puedes encontrar adentro, ¡explora las apps y descúbrelo!
